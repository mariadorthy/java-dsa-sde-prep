## How to Run

1. Clone the repository:

```bash
git clone https://github.com/mariadorthy/java-dsa-sde-prep.git
````

2. Navigate to the datatypes package:

```bash
cd src/basics/datatypes
```

3. Compile and run any file:

```bash
javac PrimitiveTypesDemo.java
java PrimitiveTypesDemo
```

---

> Each demo focuses on execution behavior to reinforce core data type concepts.
