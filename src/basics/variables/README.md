## How to Run

1. Clone the repository:

```bash
git clone https://github.com/mariadorthy/java-dsa-sde-prep.git
```

2. Navigate to the variables package:

```bash
cd src/basics/variables
```

3. Compile and run any file:

```bash
javac ScopeDemo.java
java ScopeDemo
```

---

> Each demo focuses on execution behavior to reinforce core variable concepts.
